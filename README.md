# The Gourmet Haven - Restaurant Website

![The Gourmet Haven Banner](https://source.unsplash.com/1600x400/?restaurant,fine-dining)

## Overview

Welcome to The Gourmet Haven, an elegant and responsive restaurant website designed to showcase fine dining experiences. This modern, user-friendly website is built with HTML5, CSS3, and vanilla JavaScript, featuring a clean design that highlights the restaurant's menu, reservations, and contact information.

## Customization

### Changing Content

- **Menu Items**: Update `menuData` array in `main.js` to modify menu items, prices, and categories
- **Testimonials**: Edit the `testimonials` array in `main.js` to update customer reviews
- **Contact Information**: Update contact details in `index.html`

### Styling

- **Colors**: Customize the color scheme by modifying the CSS variables in `:root` selector in `style.css`
- **Fonts**: Change fonts by updating the `@import` URL in `style.css`
- **Layout**: Adjust the grid and flexbox properties in `style.css` for different layouts
